# 心理学武器库（底层原理）

本文档包含 li-cover 使用的认知心理学和行为经济学原理。

---

## 武器1：系统1思维劫持（卡尼曼）

**原理：**
封面不是"信息"，而是**直觉触发器**。用户在"理解"之前就已经被捕获。

**应用：**
- 设计的是**刺激场**，不是文字
- 0.5秒内完成直觉判断
- 每个字都是感官刺激

---

## 武器2：存在性焦虑打击（海德格尔）

**三重脆弱点：**

**① 畏（Angst）- 无对象的存在性焦虑**
```
不是具体恐惧，而是存在性不安
"AI在骗你" > "AI可能出错"
```

**② 沉沦（Verfallen）- 常人状态依赖**
```
制造"常人"与"本真"的张力
"99%的人不知道" = 你在"常人"状态
"高手都在用" = 承诺"本真性"
```

**③ 操心（Sorge）- 时间性恐惧**
```
压缩时间性，消除不确定
"3秒"、"1招"、"立刻" = 确定感
```

---

## 武器3：身体图式抓握（梅洛-庞蒂）

**原理：**
知觉是身体的前反思"抓握"，不是认知加工。

**应用：**
- 红色 → 身体自动警觉
- 大字 → 视觉自动聚焦
- emoji → 情绪自动共鸣

---

## 武器4：损失厌恶（卡尼曼）

**选择：**
损失框架（避免失去）> 收益框架（获得好处）

**原因：**
损失的痛苦是收益快乐的2倍

**应用：**
- 优先否定词：别、停、错、假
- 制造对立：你vs他们、真相vs谎言
- 强化损失：被骗、被坑、漏钱

---

## 武器5：社会证明（西奥迪尼）

**原理：**
人们通过观察他人来决定自己的行为

**应用：**
- "99%的人不知道" → 制造稀缺感
- "高手都在用" → 从众心理
- "官方推荐" → 权威背书

---

## 武器6：稀缺性原理（西奥迪尼）

**原理：**
机会越少，价值越高

**应用：**
- 时间稀缺："最后3天"、"限时"
- 信息稀缺："内幕"、"独家"
- 认知稀缺："99%的人不知道"

---

## 武器7：信任转移机制（社会学）

**原理：**
用户不想自己判断，想借用权威的判断。点击的本质是「我相信他已经帮我筛选过了」。

**可借用的权威层级：**
- **S级：** 行业教父（吴恩达、Sam Altman、李飞飞、Karpathy）
- **A级：** 官方机构（OpenAI官方、Anthropic官方、Google官方）
- **B级：** 头部博主/大V/知名公司
- **C级：** 个人经验权威（"我亲测"、"我踩过坑"）

**应用公式：**
```
[权威人物] + [刚说/刚做的事] + [对用户的价值] + [我的解读]
```

---

## 武器8：情绪叠加策略（班杜拉自我效能理论）

**原理：**
对焦虑受众（初学者、新手）来说，单一的门槛保证不够，**多维情绪叠加**才能真正消除顾虑。

**两种"重复"的本质区别：**

**❌ 有害冗余（同一维度重复）：**
- "极简安装" + "简单上手" → 都在说操作简单，信息无增量

**✅ 有效叠加（不同维度强化）：**
- "0基础" + "小白入门" → 前者消除**能力焦虑**（我技术不够），后者触发**身份认同**（这是为我这种人准备的）
- 两个词指向同一受众，但击中不同的心理触发点

**应用场景：**
- 工具教程类、安装类 → 主动叠加门槛保证词
- 爆款验证：《从0开始安装🦞｜小白教程》证明叠加有效，不是浪费

**判断方法：**
- 问：两个词是否触发了**不同的情绪维度**？
- 是 → 有效叠加，保留
- 否 → 真正冗余，删除
