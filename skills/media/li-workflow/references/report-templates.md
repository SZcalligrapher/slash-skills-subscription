## 示例：完整优化流程执行

### 用户反馈
```
用户: "为什么AI生成内容后没有标记选题完成？"
```

### AI执行
```
1. 读取CLAUDE.md和AI能力定义.md
2. 发现问题: "生成内容"步骤缺少"标记选题完成"的执行规范
3. 生成优化方案:
   - 在CLAUDE.md的"第四步:生成内容"中新增执行规范
   - 在AI能力定义.md中新增"标记选题完成"能力
4. 执行优化:
   - 更新CLAUDE.md
   - 更新AI能力定义.md
5. 保存优化报告
6. 告知用户: "已优化流程，下次AI会自动标记选题完成"
```
