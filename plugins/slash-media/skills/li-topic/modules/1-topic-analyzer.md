# 模块1: 选题诊断 (Topic Analyzer)

## 功能定位
判断选题的爆款潜力,提供升级方案

## 核心能力
1. 判断选题层级(🚲自行车/🛵电瓶车/🚗小汽车/✈️大飞机)
2. 分析触发的用户痛点
3. 提供保守/激进两种升级方案
4. 预估流量天花板

## 输出格式

```markdown
## 🎯 选题诊断结果

**选题层级:** [🚲/🛵/🚗/✈️ X级]
**流量预估:** [X-X万]
**预期收藏/点赞比:** [X.X]（参考下方基准表，根据内容类型预判）
**预期涨粉率:** [X.X%]（参考下方基准表，根据内容类型预判）

### 核心问题
1. [问题1]
2. [问题2]

### 升级方案

#### 方案A: 保守优化
- 升级后选题: "[具体选题]"
- 核心改进: [说明]
- 流量预估: [X-X万]

#### 方案B: 激进改造
- 升级后选题: "[具体选题]"
- 核心改进: [说明]
- 流量预估: [X-X万]
- 风险提示: [说明]
```

## 判断标准

### 🚲 自行车级 (1-3万播放)
- 无明确利他价值
- 用户无动机点击

### 🛵 电瓶车级 (3-10万播放)
- 有利他价值,但无人设
- AI可替代,无粘性

### 🚗 小汽车级 (10-50万播放)
- 有利他价值 + 有人设
- 但缺少强情绪触发

### ✈️ 大飞机级 (50万+播放)
- 有利他价值 + 有人设 + 强情绪触发
- 触发用户核心痛点(焦虑/窥探/冲突)

## 收藏/点赞比参考基准（跨4位博主验证）

> 诊断时根据内容类型预判收藏/点赞比，帮助创作者选择最优内容形式。

| 收藏/点赞比 | 内容类型 | 含义 | 数据来源 |
|------------|---------|------|---------|
| >2.0 | 资源清单 / Prompt模板 | 极高实用价值，用户反复查看 | 姜学长「四句用AI快速学习」2.27、「5个视频」2.14 |
| 1.5-2.0 | 教程 / 方法论 | 高实用价值，用户收藏学习 | 黑猩猩Claude Code入门1.73、渐进式披露1.61；鲤哥ClaudeCode安装1.63 |
| 1.0-1.5 | 工具测评 / 专家访谈 | 平衡型，点赞收藏均衡 | 黑猩猩Gemini实测1.36、Gavin Baker 1.32；蜗牛椅子1.18 |
| <1.0 | 观点 / 新闻资讯 | 情绪传播型，点赞高收藏低 | 黑猩猩智能体失败0.90、品味0.85；蜗牛MacBook 0.19 |
