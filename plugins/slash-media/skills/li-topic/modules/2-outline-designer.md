# 模块2: 大纲设计 (Outline Designer)

## 功能定位
设计内容结构和情绪节奏

## 核心能力
1. 设计内容分段结构
2. 规划情绪波峰节奏
3. 设计时间节点
4. 提供文案框架(不是完整文案)

## 输出格式

```markdown
## 🎬 内容大纲 (60秒短视频)

### 1. 开头钩子 (0-5秒)

**文案框架:**
[大纲,不是完整文案]

**设计意图:**
[为什么这样设计]

---

### 2. 第一段落 (5-20秒)

**文案框架:**
[大纲]

**情绪节奏:** [平缓/上扬/冲突]

**波峰设计:**
- 第一个小高潮在[X]秒
- 触发点:[具体设计]

---

### 3. 第二段落 (20-40秒)

**文案框架:**
[大纲]

**情绪节奏:** [上扬/冲突]

**波峰设计:**
- 第二个小高潮在[X]秒
- 触发点:[具体设计]

---

### 4. 第三段落 (40-60秒)

**文案框架:**
[大纲]

**情绪节奏:** [冲突/爆发]

**波峰设计:**
- 最大高潮在[X]秒
- 触发点:[具体设计]

---

### 5. 结尾行动 (60-65秒)

**文案框架:**
[大纲]

**行动驱动:**
- 评论驱动:[具体设计]
- 关注驱动:[具体承诺]
```

## 设计原则

1. **开头3秒必须抓人** - 用具体数字/反常识观点/强痛点
2. **每15-20秒一个小高潮** - 保持用户注意力
3. **40-50秒最大高潮** - 情绪峰值+核心价值
4. **结尾必须有行动驱动** - 评论/关注/收藏

## 注意事项

- 只提供框架,不写完整文案
- 每段说明设计意图
- 标注情绪节奏和波峰位置
